💡 Стратегии по секциям

🎧 Listening 6.0 → 7.5

  • Читай вопросы ДО прослушивания — predict тип ответа
  • Разные акценты: British, Australian, American — Cambridge 13–18
  • Ежедневно в дороге: BBC 6 Minute English (Part 1–2 уровень)
  • Проверяй орфографию — частая ошибка на -1 балл
  • Фокус на S3 & S4 — academic, самые сложные
  • После теста читай транскрипты, выписывай новые фразы

📖 Reading 5.5 → 7.5

  • Skimming: первое + последнее предложение абзаца за 3 мин
  • TFNG: False = противоречит тексту, Not Given = не упоминается
  • Matching Headings: читай весь абзац → main idea → заголовок
  • Timing: строго 20 мин на текст, цель — закончить за 55 мин
  • Учи парафразы — IELTS всегда использует синонимы
  • Scanning: ищи ключевые слова, не читай всё подряд

✍️ Writing T1 5.5 → 7.0

  • Overview — самый важный параграф, пиши его 2-м (не 1-м!)
  • Выбирай 3–4 ключевых тренда, не описывай каждое число
  • Vocabulary: surge, plummet, fluctuate, plateau, remain stable
  • Сравнивай группы данных, а не перечисляй по очереди
  • 150–180 слов идеально. Больше 200 = потеря времени на T2
  • Практикуй все типы: line, bar, pie, table, map, process

📝 Writing T2 5.5 → 7.0

  • 5 мин планирование ОБЯЗАТЕЛЬНО. Никогда не пиши без структуры
  • Структура: Intro → Body 1 → Body 2 → Conclusion
  • Каждый Body: Topic sentence → Explain → Example
  • Conditionals, relative clauses, passive voice = грамматика 7+
  • 260–280 слов идеально. Varied linking words = Cohesion 7
  • Типы: Opinion, Discussion, Problem-Solution, Double Question

🗣️ Speaking 6.0 → 7.5

  • Part 1: прямой ответ + 1–2 предложения расширения, без пауз
  • Part 2: ВСЮ 1 мин подготовки — записывай 5 ключевых слов
  • Part 2 структура: Intro → Детали → Прошлое → Мнение/Чувства
  • Part 3: обобщённые ответы. "People tend to..." — не личное
  • Записывай себя каждый день — слушай и находи повторы слов
  • Idioms — только естественные, не вставные. Authenticity wins.

⚡ Режим дня и продуктивность

  • 05:40–07:00 — утренняя сессия. Свежий мозг = лучшая учёба!
  • Дневной сон 13:00–16:00 критически важен для памяти
  • Основной блок 16:00–21:00 — 5 часов структурированной учёбы
  • Тренировка (Пн–Сб) реально улучшает концентрацию
  • Pomodoro: 25 мин фокуса → 5 мин отдыха → повтор
  • Ужин в 20:00. Не учись сразу — 15 мин пауза, потом повтор
